## Setup
```
$ brew install deno
$ git clone git@github.com:tayutaedomo/deno-sandbox.git
```


## Local Server
```
$ cd deno-sandbox
$ deno run --allow-env --allow-net main.ts
$ open "http://localhost:8080"
```
